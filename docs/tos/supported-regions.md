### 🌍 NexToken支持的国家和地区 (Supported Regions & Export Compliance)

**最后更新日期：2026年5月**

由于 NexToken的所有核心基础设施与数据网关均物理部署于**美国**，为确保严格符合美国的技术合规、出口管制以及国际数据跨境传输的安全要求，本平台实施极高标准的地理访问控制（Geo-Fencing）。

在调用本平台服务之前，您必须确认您的实体运营所在地以及网络请求的发起地，均完全处于我们的支持白名单范围内。

#### 1. 获批服务区域 (Approved Regions)

本平台的 API 路由与大模型分发服务，**仅对以下国家或地区**的企业级开发者和授权实体开放：

* **北美洲：** 美国 (United States)、加拿大 (Canada)。
* **欧洲：** 欧洲联盟 (EU) 成员国、欧洲经济区 (EEA) 成员国、英国 (United Kingdom) 及瑞士 (Switzerland)。

若您的业务主体、服务器节点或流量出口不位于上述区域，您将无权获取、注册或使用本平台的 API 密钥。

#### 2. 严格受限与禁止区域 (Prohibited Regions)

除上述“获批服务区域”外，NexToken不向全球其他任何国家或地区提供网络接入服务。

**特别声明：** 为严格遵守相关区域的监管政策与阻断要求，本平台**未在亚洲地区（包括但不限于中国大陆、中国香港、中国澳门等）设立任何运营实体、代理机构或服务器节点。** 系统已在网关层级部署了严格的 IP 阻断策略，彻底拒绝来自上述受限区域的任何直连请求。

#### 3. 反规避条款与调用方绝对责任 (Anti-Circumvention)

作为调用方，您必须对所有网络请求的真实地理来源负责。本平台严禁任何形式的区域伪装与规避行为：

* **禁止技术伪装：** 严禁使用虚拟专用网络 (VPN)、匿名代理 (Proxies)、Tor 网络或任何其他 IP 伪造与流量转发技术，来掩盖您真实的地理位置并试图绕过本平台的区域拦截系统。
* **法律防火墙与单方免责：** 若您（或您的下游用户）故意隐瞒真实地理位置、使用规避技术非法接入本加拿大服务器，从而导致触发任何第三方国家/地区的数据监管审查、封禁或引发法律诉讼，**所有法律后果、连带风险及刑事赔偿责任均由您个人全权承担**。
* NexToken作为被欺骗的底层通道，依法享有完全的免责权，且不接受、不承认任何来自非支持国家/地区的调查、执法或追责要求。

#### 4. 实时风控与熔断

本平台的美国网关将实时动态监测 API 请求的来源 IP、ASN (自治系统网络) 及其路由特征。一旦系统识别到来自非支持区域的请求，或检测到明显的 VPN/代理规避特征，将立即触发自动熔断机制，永久注销该 API 密钥并拉黑相关关联信息。
