### 📌 NexToken服务特定条款 (Service Specific Terms)

**最后更新日期：2026年5月**

本《服务特定条款》针对 NexToken提供的基于大语言模型（LLM）的路由分发业务设定了具体的技术与运营细则。接入本系统即代表您知悉并同意以下针对 AI 接口特有的业务规则。

#### 1. AI 生成内容的不可预测性 (AI Hallucinations & Non-Determinism)

* **不对准确性负责：** 机器学习模型具有概率性特征，可能会生成不准确、不真实或具有误导性的输出（即“AI 幻觉”）。NexToken 作为纯粹的美国底层数据转发通道，对任何接口返回结果的真实性、准确性、完整性或适用性**不作任何担保**。
* **人工审查义务：** 您必须在您的业务流程中建立适当的“人工参与（Human-in-the-loop）”或自动化过滤机制。您在将基于本 API 生成的内容用于重大决策、医疗、法律建议或公开发布之前，必须进行严格的独立验证。

#### 2. 数据处理与零日志政策 (Zero-Log Policy)

为最大限度保护调用方隐私并遵守加拿大相关数据安全法规，本平台的网关节点执行极其严格的数据隔离策略：

* **不存储业务负载：** 当您的请求通过我们在美国的网关时，系统**仅在内存中**处理数据报文以完成向大模型供应商的转发。我们**绝对不记录、不落盘、不存储**您发送的具体提示词（Prompt）、上传的文档/图像，以及模型返回的具体生成内容（Completion）。
* **仅保留计费元数据：** 平台数据库仅保留用于计费和风控的基础元数据，包括：调用时间、消耗的 Token 数量、请求的 IP 地址、模型名称及错误状态码。

#### 3. 计费、倍率与退款策略 (Billing & Refund Policy)

* **动态倍率计费：** 本平台通过统一汇率和特定的模型倍率（Multiplier）进行积分/余额扣减。您在发起调用前有义务查阅平台公布的实时模型定价表。系统将按照实际消耗的 Token 数量或上游接口计算的计费单位进行精确扣费。
* **严苛的无退款原则：** 鉴于 API 调用的不可逆性，除因本平台核心网关自身故障导致的错扣外，其他情况**一律不予退款或积分补偿**。这包括但不限于：
* 因您发送的 Prompt 违反上游内容安全策略而导致的请求失败或封禁。
* 您对模型生成的最终内容质量不满意。
* 因您的代码逻辑错误导致的网络超时或重复请求。



#### 4. SLA 与上游依赖免责 (SLA & Upstream Dependency)

* **无 SLA 承诺：** 本服务是一个高度依赖第三方基础设施的中间件路由。我们**不提供**任何固定百分比（如 99.9%）的服务水平协议（SLA）保障。
* **级联故障免责：** 如果上游大模型供应商（如 OpenAI, Anthropic 等）发生服务器宕机、区域性网络中断、API 接口降级或速率限制策略变更，导致您无法正常调用本平台接口，NexToken 将尽可能通过切换备用路由来缓解，但对此类不可抗拒的上游级联故障**不承担任何赔偿责任**。
* **速率限制 (Rate Limits)：** 为保护加拿大网关集群的稳定性，系统会对所有 API 密钥实施动态并发限制（RPM/TPM）。我们保留根据服务器负载情况随时调整您并发上限的权利，且无需事先通知。
